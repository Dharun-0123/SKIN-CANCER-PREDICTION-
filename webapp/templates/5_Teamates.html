About the Project

SkinCare AI is a cutting-edge deep learning application that leverages advanced Convolutional Neural Networks (CNN) to classify skin lesions with high accuracy. Built with TensorFlow and Django, this system provides an accessible platform for early skin cancer detection, featuring multiple AI models, comprehensive analytics, and professional email integration for seamless user experience.

AI-Powered Analysis

Three advanced models: Basic CNN (92% accuracy) and DenseNet121 for comprehensive skin lesion classification.

Multi-Class Detection

Classifies 8 different skin conditions including melanoma, basal cell carcinoma, and benign lesions with detailed confidence scores.

Advanced Analytics

Comprehensive dashboard with prediction history, model comparison, export functionality, and detailed performance metrics.

AI Assistant

Integrated DermaGenie AI assistant powered by Perplexity API for personalized skincare advice and medical information.

Professional Email System

Verified domain email integration with Resend for welcome emails, OTP verification, and prediction notifications.

Secure & Scalable

Built with Django security best practices, user authentication, profile management, and admin dashboard for monitoring.
